# Toolkit

A collection of standalone Python 3.11+ CLI tools.

## Tools

| Tool | Description |
|------|-------------|
| `rename_files` | Renames all files in a folder by replacing spaces in filenames with underscores |
| `resize_images` | Resizes image files or all images in a directory to specified dimensions |
| `csv_to_json` | Converts a CSV file to a JSON file using the first row as column headers |
| `heic_to_jpeg` | Converts HEIC/HEIF image files to JPEG format |
| `csv_diff` | Compares two CSVs by a key column and reports shared and unique values |
